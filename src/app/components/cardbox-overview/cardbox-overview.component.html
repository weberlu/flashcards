Übersicht Karteikasten

... für jedes der 7 Fächer wird ein Rechteck gezeichnet, angefangen beim ersten Karteifach. Mit zunehmender Kapazität soll das Rechteck grösser werden. Im Inneren von jedem Rechteck wird die Anzahl abgelegter Wörter sowie die Maximalkapazität angezeigt.

[_] #1 -- 125 / 100
[_] #2 -- 50 / 150
[_] #3 -- 275 / 220
[_] #4 -- 10 / 330
[_] #5 -- 0 / 500
[_] #6 -- 0 / 750
[_] #7 -- 0 / 1250

NB: es kann sein, dass ein Fach "überläuft". Dies geschieht dann, wenn das nachfolgende Fach voll ist, d.h. der Benutzer die Wörter darin abfragt. Wörter, die nicht mehr korrekt übersetzt werden konnten, wandern danach zurück in das vorhergehende Fach. Die Kapazität ist also lediglich als Grenzwert zu verstehen, ab wann das Fach repetiert werden sollte.
